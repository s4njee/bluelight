# Bluelight

Interactive Three.js bokeh scene inspired by `DSC_1186 (1).jpeg`.

## Run locally

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Controls

- Drag to orbit
- Scroll to zoom
- `C` toggles chroma
- `V` toggles vignette
- `X` toggles pixel mode
- `Z` toggles trails
